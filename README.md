## 🌟 Signature Projects

### MetaCon PM — Enterprise Project Management
**Nextcloud App | Vue 3 + TypeScript + PHP 8.1 | Self-Hosted**

A professional project management solution natively built for Nextcloud featuring:
- 🎯 Multi-workspace system with role-based permissions (RBAC: Owner, Admin, Manager, Member, Viewer)
- 📊 Kanban boards with drag-and-drop functionality
- 📅 Calendar integration with event management
- ♿ WCAG 2.1 AA compliance & mobile-first responsive design
- 🧪 Comprehensive test coverage: Vitest + PHPUnit + Playwright E2E

**Tech Stack:**
```
Frontend:   Vue 3.5 | TypeScript 5.3 | Pinia | Vite | @nextcloud/vue
Backend:    PHP 8.1+ | Nextcloud AppFramework | MVC + Service Layer
Testing:    Vitest | PHPUnit | Playwright | PHPStan 1.10
DevOps:     Docker Compose | GitHub Actions CI/CD | Husky + lint-staged
```

→ [View Repository](https://github.com/Sebastian90Sonntag/MetaConNextCloudPlugin)
